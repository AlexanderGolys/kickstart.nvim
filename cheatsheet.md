## motions

# e [end] 
next end of word

# w [word] / b [backwards]  
jump over

# $ / 0 / ^ 
start/end/non-empty start of the line

# f / F <char> [find] 
Find next <char> in the line

# t / T <char> [find] 
Find character prior to <char>  

# ( / )
prev/next sentence

# { / }
prev/next paragraph 

# Ctrl-D [down] / Ctrl-U [up] 
half page down/up

# Ctrl-F / Ctrl-B
one page down/up

# G / g0
end/start of file





## modes switching

# i / a
insert before/after
